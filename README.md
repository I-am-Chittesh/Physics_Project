# Quantum States of Light – Wigner Function Visualization

## Project Overview

This project visualizes quantum states of light using Wigner functions in phase space. It demonstrates both classical and non-classical behavior using Coherent and Fock states.

## Key Concepts

* Phase Space (Position X & Momentum P)
* Wigner Function (Quasi-probability distribution)
* Non-classicality (Negative regions)

## Features

* 3D Surface Plot of Wigner Function
* Highlighting of Negative Regions
* Density Matrix Generation
* Interactive Slider for Coherent State
* Animation of State Evolution

## States Implemented

* Coherent State (Classical-like)
* Fock State (Quantum)

## Results

* Coherent states show smooth Gaussian distribution
* Fock states show negative regions → proof of non-classical behavior

##  How to Run

Install dependencies:
pip install -r requirements.txt

Run main program:
python main.py

Interactive mode:
python interactive.py

Animation:
python animation.py
